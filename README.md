## Automated-Data-Cleaning-Tool
Project Overview This project is a Python-based automated data cleaning tool designed to clean messy CSV files commonly found in real-world data analysis tasks.  It handles missing values, duplicate records, and incorrect data types using the Pandas library.  ---


---

##   Features
- Reads messy CSV files
- Removes duplicate rows
- Handles missing (null) values
- Fixes incorrect data types
- Outputs a cleaned CSV file

---

##  Technologies Used
- Python
- Pandas
